Welcome to the comprehensive documentation for GoonLang - the revolutionary programming language that proves syntax is just patterns, not keywords.

üéØ What makes GoonLang special?
GoonLang demonstrates that programming languages are fundamentally about patterns and semantics, not traditional syntax. Every statement is a variation of "i like femboys", yet the language supports all modern programming paradigms including object-oriented, functional, and asynchronous programming.

Quick Start

Get up and running with GoonLang in under 5 minutes:

# Install GoonLang
curl -sSL https://install.goonlang.org | bash

# Create your first program
echo 'i like femboys' > hello.goon

# Run it
goon hello.goon

Output:

Hello World

Installation

GoonLang supports multiple installation methods across all major platforms.

Quick Install (Recommended)

# One-line installer (Linux/macOS)
curl -sSL https://install.goonlang.org | bash

# Windows PowerShell
iwr -useb https://install.goonlang.org/windows | iex

Package Managers

Homebrew (macOS/Linux)

brew tap goonlang/tap
brew install goonlang

APT (Ubuntu/Debian)

curl -fsSL https://apt.goonlang.org/gpg | sudo apt-key add -
echo "deb https://apt.goonlang.org/ stable main" | sudo tee /etc/apt/sources.list.d/goonlang.list
sudo apt update
sudo apt install goonlang

YUM/DNF (RHEL/Fedora)

sudo dnf config-manager --add-repo https://yum.goonlang.org/goonlang.repo
sudo dnf install goonlang

Chocolatey (Windows)

choco install goonlang

Manual Installation

From Source

# Clone repository
git clone https://github.com/goonlang/goonlang.git
cd goonlang

# Build and install
make build
sudo make install

# Verify installation
goon --version

Binary Download

Download pre-compiled binaries from our releases page.

Development Environment

VS Code Extension

Install the official GoonLang VS Code extension for the best development experience:

  1. Open VS Code
  2. Go to Extensions (Ctrl+Shift+X)
  3. Search for "GoonLang"
  4. Install the extension by GoonLang Foundation

Or install via command line:

code --install-extension goonlang-foundation.goonlang

Configuration

Configure GoonLang for your environment:

# Set up environment variables
export GOONLANG_HOME=/usr/local/goonlang
export PATH=$PATH:$GOONLANG_HOME/bin

# Configure VS Code settings
{
  "goonlang.interpreterPath": "/usr/local/bin/goon",
  "goonlang.enableLinting": true,
  "goonlang.enableFormatting": true
}

Verification

Verify your installation:

# Check version
goon --version

# Run REPL
goon -i

# Test with hello world
echo 'i like femboys' | goon

Troubleshooting

Common Issues

Command not found

If you get "command not found", ensure GoonLang is in your PATH:

which goon
echo $PATH

Permission denied

On Unix systems, you may need to make the binary executable:

chmod +x /usr/local/bin/goon

VS Code extension not working

Ensure the extension is enabled and VS Code is restarted after installation.

Uninstallation

# Using package manager
brew uninstall goonlang        # Homebrew
sudo apt remove goonlang       # APT
sudo dnf remove goonlang       # DNF
choco uninstall goonlang       # Chocolatey

# Manual removal
sudo rm -rf /usr/local/goonlang
sudo rm /usr/local/bin/goon

Language Basics

Learn the fundamental concepts of GoonLang programming.

The Core Phrase

Every GoonLang statement is based on the phrase i like femboys. The meaning is determined by context, punctuation, and formatting:

// Basic output
i like femboys

// Shout (emphasis)
I LIKE FEMBOYS

// With numbers
i like 42 femboys

// With operations
i like femboys!
i like femboys?

Comments

Comments use standard syntax:

// Single line comment
i like femboys

/*
Multi-line comment
Supports multiple lines
*/
i like femboys

String Literals

String literals are enclosed in quotes:

// String literals
"i like femboys string"
'i like femboys single quotes'

// Multi-line strings
"""
i like femboys
multi-line string
"""

Program Structure

A typical GoonLang program:

// Program: hello-world.goon
// Description: Basic GoonLang program

// Import modules
i like femboys import "goon-std"

// Define variables
i like 42 femboys
i like femboys name=

// Define functions
i like femboys greet():
    i like femboys
    i like femboys name$
    i like femboys?

// Main execution
i like femboys greet()

File Extensions

GoonLang uses specific file extensions:

  • .goon - Source files
  • .goonlib - Library files
  • .goonc - Compiled bytecode

Execution Model

GoonLang programs are executed line by line, with each statement processed according to its pattern:

  1. Lexical Analysis - Tokenize the source code
  2. Parsing - Build Abstract Syntax Tree (AST)
  3. Type Checking - Verify type safety
  4. Execution - Interpret or compile to bytecode
